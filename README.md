# Electronic-Voting-Machine
Electronic voting system simulation using digital logic design (LogicWorks)
🗳️ Electronic Voting Machine (DLD Project)
📌 Overview

This project is a Digital Logic Design (DLD) simulation of an Electronic Voting Machine (EVM) developed in LogicWorks. The system allows voters to cast a vote for one of four candidates using dedicated push buttons while ensuring that only one vote can be registered at a time.

The machine keeps track of votes for each candidate, displays the final results when requested, announces the winner, and automatically detects tie situations.

🎯 Project Objectives
Allow voting for one of four candidates.
Ensure that only one vote can be cast at a time.
Count votes for each candidate separately.
Display the total votes received by each candidate.
Determine the winning candidate automatically.
Indicate a draw if multiple candidates receive the same highest vote count.

⚙️ How It Works
A voter selects a candidate using the corresponding vote button.
The system validates the input and prevents multiple votes from being registered simultaneously.
Votes are counted and stored separately for each candidate.
When the Result button is pressed, the system displays the vote counts.
The candidate with the highest vote count is declared the winner.
If two or more candidates receive the same highest vote count, the system indicates a draw.
✨ Key Features
🗳️ Single-vote validation
📊 Separate vote counters for each candidate
🔄 Reset functionality for starting a new election
🏆 Automatic winner determination
🤝 Tie detection and indication
⚡ LogicWorks-based simulation
